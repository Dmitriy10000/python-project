# Python web chat
## Гайд по запуску
- Скачать зависимости:
```
pip install -r requirements.txt
```
- Создать файл `DB_cfg.json` в корне проекта и заполнить его содержимое.
  
  Пример:
```
{
	"HOST": "localhost",
	"PORT": "5432",
	"USER": "postgres",
	"PASSWORD": "postgres",
	"DB": "Python-project"
}
```
- Запустить файл app.py
